## 关于本站

这个博客是我自己搭建的，技术栈很轻：

- 纯前端 **SPA 架构**，通过 `?p=` 参数路由
- **marked.js** — Markdown → HTML 渲染
- **GitHub Pages** 部署，写完 push 就上线
- 支持<span class="sk-ann blue">亮色</span>/<span class="sk-ann red">暗色</span>主题切换
- 配了<span class="sk-ann orange">小黑手绘插画</span>——一个认真做事的怪诞角色（一共 14 张！）

所有文章以 Markdown 格式存储，客户端实时渲染。没有数据库，没有后端，极简主义。
