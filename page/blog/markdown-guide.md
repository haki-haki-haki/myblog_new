# Markdown 写作指南

Markdown 是一种轻量级标记语言，非常适合博客写作。

## 基本语法

### 标题

使用 `#` 号可以创建标题：

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 代码

行内代码使用反引号：`console.log()`

代码块使用三个反引号：

```python
def hello():
    print("Hello, Markdown!")
```

### 链接与图片

```markdown
[链接文字](https://example.com)
![图片描述](image.jpg)
```

### 表格

| 语法 | 效果 |
|------|------|
| `**粗体**` | **粗体** |
| `*斜体*` | *斜体* |
| `` `代码` `` | `代码` |

Markdown 简单易用，是技术写作的首选格式。
